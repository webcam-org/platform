# First-Time User Journey

1.  **Download app:** The user downloads the app from the App Store or Google Play Store.
2.  **Onboarding:** The user is presented with a series of welcome screens that highlight the key features of the app.
3.  **Sign Up:** The user creates a new account.
4.  **Permissions:** The app requests permissions for notifications and camera access.
5.  **Add First Camera:** The user is guided through the process of adding their first camera.
6.  **First Notification:** The user receives their first notification when motion is detected by their camera.
