# PTZ Control Interaction

1.  **Accessing Controls:** In the Live Camera View, the user can tap the screen to reveal the PTZ controls.
2.  **Directional Pad:**
    - A directional pad with up, down, left, and right arrows is displayed on the screen.
    - The user can tap an arrow to move the camera in the corresponding direction.
    - The user can also press and hold an arrow for continuous movement.
3.  **Zoom Slider:**
    - A slider with "+" and "-" icons is displayed on the screen.
    - The user can drag the slider to zoom in and out.
4.  **Haptic Feedback:** Haptic feedback is provided when the user taps a PTZ control.
