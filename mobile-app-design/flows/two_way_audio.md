# Two-Way Audio Flow

1.  **Initiating Audio:** In the Live Camera View, the user can tap a microphone icon to initiate two-way audio.
2.  **Push-to-Talk:**
    - A "Talk" button is displayed on the screen.
    - The user presses and holds the "Talk" button to speak.
    - A visual indicator (e.g., a glowing ring around the button) is displayed while the microphone is active.
    - When the user releases the button, the microphone is muted.
