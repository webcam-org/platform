# Event Timeline Screen

**Layout:**

- A chronological list of event cards, grouped by date.

**Components:**

- **Event Card:**
  - **Thumbnail:** A thumbnail image of the event.
  - **Event Type:** An icon and text indicating the type of event (e.g., "Person detected", "Motion detected").
  - **Camera Name:** The name of the camera that detected the event.
  - **Timestamp:** The time the event occurred.

**Interactions:**

- **Tap on an event card:** Opens the recorded video clip of the event.
